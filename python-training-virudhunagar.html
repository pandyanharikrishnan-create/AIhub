Specialization

Python for Data Science and ML

Our training goes beyond basic syntax. We teach you how to use Python as a powerful tool for analyzing large datasets and training machine learning algorithms. You will learn to use Pandas to filter, group, and merge tables, and perform exploratory data analysis (EDA).

You will explore scikit-learn to split data, load standard algorithms (like linear regressions and decision trees), train models, and measure accuracy metrics. This practical training ensures you are fully prepared for advanced analytical tasks.

  • Filter, clean, and merge tables with Pandas
  • Calculate statistical descriptors using NumPy
  • Build data visualizations with Matplotlib & Seaborn
  • Implement basic machine learning models using scikit-learn
  • Interactive CLI Calculator: Build a terminal-based calculator utilizing modular functions.
  • E-commerce Web Scraper: Write a script that extracts product prices and reviews from online shops.
  • Desktop Organizer Script: Automate organizing your downloads folder by file types.
  • Data Analysis Dashboard: Create a dashboard that processes CSV data and displays charts.
